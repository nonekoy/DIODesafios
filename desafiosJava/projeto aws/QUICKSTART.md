# 🚀 Executar em 3 Passos

## 1️⃣ Instalar
```bash
npm install
```

## 2️⃣ Executar
```bash
npm start
```

## 3️⃣ Acessar
**http://localhost:3000**

---

## 🎯 O que você verá

- 📊 **Dashboard** com estatísticas
- 💊 **Produtos** - Lista e cadastro
- 📦 **Estoque** - Controle de quantidade
- 💰 **Vendas** - Processar vendas
- 📈 **Relatórios** - Análise de vendas

## 🌐 Deploy AWS

```bash
chmod +x scripts/deploy.sh
./scripts/deploy.sh
```

## 📁 Estrutura

```
farmacia-aws/
├── frontend/     # Interface web

├── scripts/      # Deploy AWS
└── docs/         # Documentação
```
